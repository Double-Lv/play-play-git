## 提交文件到本地仓库

### 实验一
1. 创建a.js
2. `git add . //添加到暂存区`
3. `git commit -m "新增文件a.js“ //提交到本地仓库，并附加信息`
4. `git status //查看状态，nothing to commit...表示没有需提交的了`

### 实验二
1. 修改a.js的内容
2. 创建b.js
3. `git add b.js //只添加b.js到暂存区`
4. `git commit -a -m "commit two files" //提交到本地仓库`
5. `git status //查看状态，a.js和b.js均被添加到了本地仓库`
